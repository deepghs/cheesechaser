Installation
===================

cheesechaser is currently hosted on PyPI. It required python >= 3.8.

You can simply install cheesechaser from PyPI with the following command:

.. code:: shell

    pip install cheesechaser

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/deepghs/cheesechaser@main

script:
